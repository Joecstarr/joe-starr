## TeX

Generates to a PDF targeting an iPad.
PDF has one page of definition name followed by a page of the actual definition content.

## Anki

`anki.py` is a python script, it takes in a .tex file (drag and drop).
It then parses out the review content and creates an [anki](https://ankiweb.net/) deck.
Import the deck into Anki desktop to review with Anki Web.
